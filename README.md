# Setup

Aottg2 folder is generally located in `Documents/Aottg2` on windows or `~/.local/share/Aottg2` on linux.

1. Save `Blub Blub.txt` in `Aottg2/CustomLogic`
2. Save `Map_Akina.txt` in `Aottg2/PersistentData`
3. Optionally save the `Leaderboard_*` files in `Aottg2/PersistentData`
4. Copy the contents of `Preset.txt` and import it for lobby game settings.
